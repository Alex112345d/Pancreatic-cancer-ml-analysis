# Pancreatic-cancer-ml-analysis
Machine learning project predicting pancreatic cancer risk using real-world clinical data, comparing Logistic Regression, Decision Tree, and Random Forest models with 10-fold cross-validation.

This project applies machine learning to predict pancreatic cancer risk, developed as part of the CET140 module assignment. Using a real-world (non-synthetic + synthetic) dataset of 50,000 patient records sourced from Kaggle, the project builds and evaluates three classification models — Logistic Regression, Decision Tree, and Random Forest — with 10-fold cross-validation used to assess model performance and generalizability. The analysis was conducted in Google Colab.

Tech used: Python, Google Colab, scikit-learn (Logistic Regression, Decision Tree, Random Forest), Pandas
Includes: Data preprocessing, model training & evaluation, cross-validation results
